Linux System Administrator

Apptricity Corporation is looking for a talented Linux System Administrator to join our growing team of IT professionals to lead the charge of software installations and upgrades to operating systems and layered software packages. The primary focus will be to provide a highly available, consistent, and resilient operating environment for client applications. This includes system administration, engineering design, and support responsibilities.

The ideal candidate will have SME level Linux knowledge and experience and can work with all internal customers from the CEO to the receptionist with ease and finesse.

Responsibilities:

  • Responsible for the installation, configuration and maintenance of Linux servers in a virtualized environment
  • Install, configure, and tune Java Application servers including: Tomcat, JBoss, WebLogic, and Websphere
  • Operate in a production data center environment with minimal supervision
  • Installation and configuration of both commercial and open source software
  • Troubleshoot Java applications on Linux and Windows
  • Troubleshoot system hardware provide failure analysis and recovery
  • Troubleshoot network issues related to WAN and LAN access to Java applications and other network accessible services (SFTP, AS2, DNS, WINS, SSH, VPN, Mail, Apache)
  • Installation and maintenance of database servers (Microsoft SQL Server, Oracle, MySQL)
  • Interface directly and indirectly with client staff and 3rd party vendors both technical and non-technical
  • Creation of new virtual machines of any OS in a Linux based host environment

Minimum Requirements:

  • 7 years of professional experience working in a client facing Production environment
  • 5 - 7 years of professional experience with Red Hat, CentOS or Fedora Linux
  • 2 - 3 years of experience with Red Hat, KVM or QEMU based virtual machines
  • 2 - 3 years of experience with kickstart/PXE boot
  • 3 - 4 years of experience with Java Application servers
  • 3 - 5 years of experience with Apache web server
  • 2 - 4 years of scripting (Perl, Bash)

Desired Skills:

  • 3 – 5 years of Windows Server experience including domain administration
  • 1 – 3 years of experience running MySQL Clustered databases
  • Network monitoring experience

Permanent Residency Required. U.S. citizenship may be required for security clearance.
