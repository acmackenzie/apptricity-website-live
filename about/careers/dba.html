Database Administrator

Apptricity is looking for a talented and agile Database Administrator to help us develop, maintain, and monitor databases in our heterogeneous database environment. We expect this individual to not only be able to handle database administration tasks but be able to participate in our projects as a key contributor. This position requires a self-motivated individual with a strong DBA foundation who can implement innovative ways to solve complex problems.

Job Responsibilities:

  • Own the data application layer of our solutions: Understand the technologies in use and how our applications talk with the database. Lead modeling activities and be able to work with Architecture and Development Leads to solve problems. Own both the logical and physical database design.
  • Own database performance: Take the lead on developing performance benchmarks of our databases, work with Architecture and IT on performance testing activities, perform SQL monitoring and optimization.
  • Own the data: Own and manage all data. Synchronize data across multiple development, testing, and production environments including migrating between disparate databases.
  • Strive for clean database design: Implement and enforce consistent database conventions. Design the best data model for the requirement at hand, be willing to normalize and de-normalize where needed.
  • Be involved and interested in application design: Be able to work with developers in solving problems as it pertains to application design, data modeling, data access, and reporting.
  • Development process and practices: Understand and know how to use version control for all databases, drive databases changes through the development process and environments, create automated ways to promote database modifications, create deployment scripts for upgrades, development and implement database standards and procedures.
  • Debug and troubleshooting: Help troubleshoot and resolve database related issues and be able to tune the application and database to improve performance.
  • Be interested: We're not just looking for someone that “can” be a DBA we're looking for someone that “likes” being a DBA. We run a heterogeneous database environment and that means we need someone that likes learning new things. Experience or interests in different Relational Databases and NoSQL Data Stores is a plus.
  • Be a leader: Lead by example and be able to effectively communicate with all types, direct and mentor team of Database Administrators. Understand that we have deadlines and lead efforts to meet them.

Requirements:

  • Bachelor's Degree in Computer Science or Information Systems Management
  • 3-5 years DBA experience
  • Must be a SQL Ninja
  • Working experience with Oracle, SQL Server and MySQL
  • Familiarity with Windows and Linux operating environments
  • Ability to learn quickly in a fast-paced dynamic environment
  • Experience with scripting languages: bash, python or groovy

Desired Requirements:

  • Experience with databases running in virtualized environments a plus
  • Experience with Postgres, MongoDB and JasperServer a plus
  • Experience supporting databases where clients are Java based applications a plus
