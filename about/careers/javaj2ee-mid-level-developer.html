Java/J2EE (Mid-level) Developer

Apptricity Corporation is adding additional Java J2EE developers to expand its product suite. Positions include JSP front end focus as well as EJB backend developers. Apptricity produces leading edge software utilizing state of the art technology.

Responsibilities:

  • Assist in development of the Apptricity application interface and tool set using Java tools and technologies.
  • Become knowledgeable in the Software/Hardware architecture of Apptricity products, including system configuration, database schema, functions, indexes, and views.
  • Work within project teams and customer service teams as a technical resource and contribute to successful implementations and customer satisfaction.
  • Work in a team environment to maximize customer satisfaction to develop strong customer references.
  • Develop detailed knowledge of all Apptricity products, product architecture, development tools and common customer operating environments.
  • Provide assistance with testing and problem resolution as may be required.

Required Skills:

  • University degree in Computer Science or related field.
  • Minimum of 3 years’ experience working with Java based technologies.
  • Current Java (J2EE) and back-end development.
  • Permanent Residency Required. U.S. citizenship may be required for security clearance.

Desired Skills:

  • Experience using SQL, SQL Server, Oracle, or MySQL.
  • Experience developing commercial transaction based financial applications.
  • Experience with applications that function on multiple browsers, platforms and versions.
  • Experience with Hibernate, EJB/Spring, Jersey and/or CXF
  • Excellent communication skills, both verbal and written.
  • Excellent problem solving abilities.
